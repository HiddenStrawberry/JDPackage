京东抽奖爬虫 LiteVersion1
===================
这段代码用来爬取整个京东的LotteryCode抽奖代码，你可以通过这些抽奖代码进行抽奖。


----------


使用方法
----
----------
直接运行LiteV1.py，可以得到一个字典结构，key为lc,value为活动Sale页面
